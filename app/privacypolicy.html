Privacy Policy: DFYU Barrel Discord Bot

Updated on June 9, 2025

DFYU Barrel is a Discord bot created for entertainment and engagement within the DFYU community. We take your privacy seriously. This page outlines how DFYU Barrel handles data.

1. Data Collection

DFYU Barrel does not collect, log, or store any personal user data. This includes but is not limited to:

Discord usernames or IDs

Messages or conversations

Server or channel identifiers

IP addresses or any identifiable user data

2. Bot Functionality

DFYU Barrel uses Gemini AI to generate intelligent responses for certain commands. These AI requests are processed in real-time and discarded after use. We do not log these inputs or outputs.

3. Use of External APIs

Certain fun commands like hug, kiss, etc., retrieve images or GIFs from third-party APIs. These APIs process minimal necessary information to serve a response, and no data is stored or saved by DFYU Barrel.

4. No Tracking or Analytics

DFYU Barrel does not implement any tracking, cookies, or analytics systems.
